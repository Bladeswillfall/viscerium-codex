This page provides the detailed Okse calendar route used by date badges:

```text
/calendar/okse/
```

Date badges link here with anchors such as:

```text
/calendar/okse/#solmanuthur-16
/calendar/okse/#engimanutur-02
```

<CalendarYear calendarId={"okse"} year={4} />

The rendered year above is a leap year so both Engimanutur days are visible.

For the public overview, see the main [Calendar](/calendar/) page.
