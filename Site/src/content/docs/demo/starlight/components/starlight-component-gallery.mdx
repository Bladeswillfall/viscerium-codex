## Calendar components

The full Okse calendar is now placed with a shortcode, not automatically injected into the page title area. This allows the calendar to appear wherever the note author places it.

```md
[Calendar:ID-0001]
```

<CalendarYear calendarId={"okse"} year={4} />

The page also declares an intercalary leap-day `calendarDate`, so the automatic date badge should show `Engimanutur 02` for year `20004`.

## Typography system

The codex uses **Cinzel** as the display face for the main title layer, **Source Serif 4** for long-form prose, **IBM Plex Sans** for interface and metadata, and **IBM Plex Mono** for code or technical fragments.

## Mathematical notation

Inline TeX can be written like GitHub-style Markdown: $a^2+b^2=c^2$.

Display TeX can be written with double-dollar delimiters:

$$
a^2+b^2=c^2
