## Steps

<Steps>

1. Copy the Markdown demo into `Vault/Lore/Demo/Starlight Feature Demo.md`.

2. Copy this MDX page into `Vault/Lore/Demo/Starlight Component Gallery.mdx`.

3. Run the sync and build commands.

   ```bash title="Build test"
   cd Site
   npm run sync
   npm run build
   ```

4. Check the public site and delete the demo pages once styling is stable.

</Steps>
