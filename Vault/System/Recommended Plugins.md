# Recommended Plugins

The publishing workflow does not require paid plugins. Start simple, then add optional plugins only if they help your writing.

## Recommended actions

1. Enable the built-in **Templates** plugin and set `Vault/Templates/` as the template folder.
2. Optional: install **Templater** for prompts, automatic dates, and generated slugs.
3. Optional: install **Dataview** for private index/dashboard notes in `Vault/System/`.
4. Optional: install **Advanced Tables** if you want table-style infobox editing.
5. Optional: install **Editing Toolbar** if you prefer a more visual editing experience.
6. Optional: install **Style Settings** and a theme you like for Obsidian-only appearance changes.

## Publishing caution

Dataview queries, plugin-only syntax, and theme-specific styling may not render the same way on the Astro site. For public pages, write important content as normal Markdown and keep required publishing metadata in frontmatter.
